# archaeology-ml-classification
Machine learning classification of archaeological samples using geochemical and spectral data (PCA, Random Forest, KNN)

## Project Overview
This project demonstrates an end-to-end machine learning pipeline for classifying archaeological samples using tabular geochemical and spectral data.

The notebook implements data preprocessing, dimensionality reduction, and supervised classification models to explore pattern recognition and predictive modelling in complex scientific datasets.

This project serves as a demo of applied machine learning workflows relevant to recommendation, search, and user modelling tasks.

---

## Methods
The following techniques were applied:

- Data cleaning and preprocessing
- Feature scaling (StandardScaler)
- Dimensionality reduction (PCA)
- K-Nearest Neighbors (KNN)
- Random Forest classification
- Model evaluation (Precision, Recall, F1-score)
- Confusion matrix visualization
- Feature importance analysis

---

## Results
- PCA provided an interpretable low-dimensional representation of class separability
- Random Forest achieved strong classification performance on tabular data
- Feature importance analysis highlighted key predictive variables
